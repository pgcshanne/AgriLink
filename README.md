# AgriLink

## Design and Implementation of an IT-Based Solution for Improving Agricultural Productivity among Smallholder Farmers in Bogo City

### Description
AgriLink is a Flutter-based mobile application developed to assist smallholder farmers in Bogo City through intelligent agricultural technologies. The application integrates Artificial Intelligence (AI) to provide crop disease detection, soil analysis, localized weather monitoring, planting and harvest recommendations, crop growth tracking, predictive market price information, offline functionality, and multilingual support.

### Features
- AI-powered crop disease detection
- AI-based soil analysis and recommendations
- AI-based planting and harvest advisory
- Crop growth tracking
- Localized weather monitoring
- Predictive market price costing
- Offline functionality
- Multilingual support
- Farmer feedback
- Administrator dashboard

### Technologies Used
- Flutter
- Dart
- PHP
- MySQL
- XAMPP
- Visual Studio Code

Agile Software Development Life Cycle (SDLC)
